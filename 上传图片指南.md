# 如何上传图片到这个仓库

本仓库用于存储毕设论文相关图片。以下是三种简单快速的上传方法，**推荐使用方法一**（最简单）。

## 方法一：使用 GitHub 网页界面上传（最简单，推荐）

这是最简单快速的方法，不需要安装任何软件。

### 步骤：

1. **打开仓库网页**
   - 访问：https://github.com/tuanziya666/lunwen

2. **进入上传页面**
   - 点击页面上方的 `Add file` 按钮
   - 选择 `Upload files`

3. **上传图片**
   - 将你的 20 张图片拖拽到上传区域
   - 或点击 `choose your files` 选择文件

4. **提交上传**
   - 在页面底部填写提交信息（例如：`上传论文图片`）
   - 点击 `Commit changes` 按钮

✅ **完成！** 图片已经上传到仓库了。

---

## 方法二：使用 Git 命令行上传

如果你熟悉命令行操作，这个方法也很快捷。

### 前提条件：
- 已安装 Git
- 已配置 GitHub 账号

### 步骤：

1. **克隆仓库到本地**
   ```bash
   git clone https://github.com/tuanziya666/lunwen.git
   cd lunwen
   ```

2. **创建图片文件夹（可选）**
   ```bash
   mkdir images
   ```

3. **复制图片到仓库**
   - 将你的 20 张图片复制到 `images` 文件夹中
   - 或直接复制到仓库根目录

4. **提交并推送**
   ```bash
   git add .
   git commit -m "上传论文图片"
   git push
   ```

✅ **完成！**

---

## 方法三：使用 GitHub Desktop（图形化界面）

适合不熟悉命令行但想在本地管理文件的用户。

### 前提条件：
- 下载并安装 [GitHub Desktop](https://desktop.github.com/)

### 步骤：

1. **克隆仓库**
   - 打开 GitHub Desktop
   - File → Clone Repository
   - 选择 `tuanziya666/lunwen`
   - 选择保存位置，点击 Clone

2. **添加图片**
   - 在文件管理器中打开仓库文件夹
   - 将 20 张图片复制到文件夹中

3. **提交更改**
   - 回到 GitHub Desktop
   - 在左侧可以看到新添加的文件
   - 在底部填写提交信息（例如：`上传论文图片`）
   - 点击 `Commit to main`

4. **推送到 GitHub**
   - 点击顶部的 `Push origin` 按钮

✅ **完成！**

## 建议的文件夹结构

为了更好地组织图片，建议使用以下结构：

```
lunwen/
├── images/           # 图片文件夹
│   ├── 图1-xxx.png
│   ├── 图2-xxx.jpg
│   └── ...
└── README.md
```

## 支持的图片格式

- PNG (.png)
- JPEG (.jpg, .jpeg)
- GIF (.gif)
- SVG (.svg)
- 其他常见图片格式

## 注意事项

1. **文件大小限制**：单个文件不要超过 100MB
2. **文件命名**：建议使用有意义的文件名，如 `图1-系统架构.png`
3. **图片总数**：本仓库需要上传 20 张图片
4. **网络问题**：如果上传较大文件失败，建议使用方法一分批上传

## 常见问题

### Q: 上传后如何查看图片？
A: 在 GitHub 仓库页面中直接点击图片文件名即可查看。

### Q: 如何在论文中引用这些图片？
A: 点击图片，然后点击 `Raw` 按钮，复制 URL 地址即可使用。

### Q: 上传错误怎么办？
A: 可以在 GitHub 网页界面中直接删除文件，然后重新上传。

---

**推荐**：如果你只是想快速上传 20 张图片，直接使用**方法一**（网页上传）最简单！
